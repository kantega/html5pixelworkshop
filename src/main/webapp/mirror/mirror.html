Mirror, mirror on the screen

Your goal in this task is to create a simple video player capable of:

  • Playing the video file /videos/video.mp4
  • Playing and pausing via custom buttons
  • Showing the progress of the playing video
  • CSS: Make the play/pause buttons show a "hand" cursor on mouse over, to indicate they are clickable
  • Can you show the pause button only when the video is playing?

This HTML file is found in src/main/webapp/simpleplayer/simpleplayer.html You will also need to edit Javascript in mirror.js in the same folder and perhaps some CSS in simpleplayer.css
