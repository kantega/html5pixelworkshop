A simple video player

Your goal in this task is to create a simple video player capable of:

  • Starting and pausing playback by listening to button click events
  • Showing the progress of the playing video
  • CSS: Make the play/pause buttons show a "hand" cursor on mouse over, to indicate they are clickable
  • Can you show the pause button only when the video is playing?

This HTML file is found in src/main/webapp/simpleplayer/simpleplayer.html You will also need to edit Javascript in simpleplayer.js in the same folder and perhaps some CSS in simpleplayer.css (The code will contain TODO comments with some additional hints)
